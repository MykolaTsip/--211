**Використати мініму 6 видів селекторів в css

Завдання 1: Зміна кольору елемента
Створіть HTML-файл з прямокутником (наприклад, <div>) і використайте CSS для зміни кольору цього прямокутника 
з початкового кольору на інший кольор по наведенню на нього.

Завдання 2: Проста анімація переміщення
Створіть HTML-файл з блоком (наприклад, <div>), який буде представляти об'єкт, що рухається коли наводите курсор миші на нього. 

Завдання 3: Анімація ключових кадрів
Створіть HTML-файл з об'єктом (наприклад, <div>), зробіть, щоб елемент змінював розмір та кольори при завантаженні сторінки.
підказка: (keyframes)

Завдання 4: Зміна фону з анімацією

Створіть HTML-файл з об'єктом (наприклад, <div>), і використайте CSS для створення анімації, 
яка буде змінювати фон цього об'єкта на інший фон з плавною анімацією.
підказка: (transition)

Завдання 5: Ротація об'єкта
Створіть HTML-файл з об'єктом (наприклад, <div>), і використайте CSS для 
створення анімації ротації цього об'єкта навколо своєї власної осі.
